## 후기
이번 주의 공통문제들은 모두 풀어본 문제였으나 오래되어 그런지 다시 이해하고 풀어봐야 했다.
이해하는 과정에서 이전 풀이에 설명이 있었으면 쉽게 넘어갈 수 있었으나 주석을 달지 않아 좀 어려웠다.
앞으로 풀이에 설명을 열심히 쓰도록 하자.

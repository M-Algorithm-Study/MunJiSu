## 후기
이번 주의 공통문제인 색종이와 가위, A와 B 모두 생각하기가 어려웠다.
A와 B는 많은 생각을 하지 않고, 문제의 순서에만 따라서 풀었더니 시간초과가 발생했고, 생각과는 반대로 문제를 풀어야 했다.
색종이와 가위는 가위질을 해서 만들 수 있는 색종이의 개수를 파악하는 부분에서 시간이 오래걸렸던 것 같다.
아직도 너무나도 많이 부족한 것 같다. 열심히 공부해야겠다.
